# Agentic Frameworks Research 2023-2026

This reference map covers the transition from prompt-only systems to orchestrated agent frameworks.

## Single-Agent to Multi-Agent Progression

1. CAMEL and AutoGen introduced explicit agent-role communication as a first-class mechanism.
2. MetaGPT extended this into software-style team workflows with specialized roles.
3. Voyager showed long-horizon skill accumulation in embodied settings.

## Key Design Dimensions

1. Role decomposition and communication protocol.
2. Memory design and retrieval policy.
3. Tool interface and action safety constraints.
4. Feedback and optimization loop design.

## 2025+ Direction

Recent work emphasizes training and optimization of agent behavior, not just handcrafted prompting. This shifts effort toward measurable policy learning loops.

## Practical Mapping in This Wiki

These papers inform:

1. ReAct and planning prompts.
2. Multi-step workflow orchestration patterns.
3. Agentic evaluation harness design.
